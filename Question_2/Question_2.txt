Evernote is considered as a software as a Service(SaaS) in which software is hosted on Cloud-based platform or delivered software online for the end users. Users can access the application via web browser or mobile apps without even installing the software on user's computer, and data such as documents stored on evernote's server can be accessed from any location with the help of internet connection. In Addition to this, users can pay additional fees to unlock more features of Evernote.
